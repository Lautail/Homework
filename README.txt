Q-talk网上论坛
  Q-talk网上论坛分为Q-talk论坛和Q-talkCMS管理系统。其中，Q-talk论坛包含用户登录注册，用户个人中心，轮播图，帖子查看，评论帖子，以及帖子推送。
Q-talkCMS管理系统包括CMS用户个人中心，帖子管理，评论管理，Q-talk论坛用户管理，CMS用户以及组管理和简单的Q-talk论坛数据分析。
  帖子推送模块分为帖子排行和帖子推荐，帖子排行根据帖子的阅读数和评论数以及发帖时间来进行排行。帖子推荐根据用户评论帖子的记录，利用基于物品的
协同过滤算法来进行推荐。当用户没有评论行为时，推荐为帖子排行中排名较高的帖子，而当用户的评论行为较少，基于物品的协同过滤有可能的不到结果，此
时推荐的帖子也为排名较高的帖子。
  利用第三方实现的TextRank算法的工具包，对帖子的内容生成摘要。
